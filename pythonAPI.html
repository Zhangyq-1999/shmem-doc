SHMEM对外接口

  1. 初始化共享内存模块

    def shmem_init(mype, npes, mem_size) -> int

    参数/返回值

    含义

    mype

    本地处理单元索引，范围在 0 ~ npes 之间

    npes

    处理单元总数

    mem_size

    每个处理单元的内存大小，以字节为单位

    返回值

    成功返回0，其他为错误码

  2. 完成共享内存模块

    def shmem_finalize() -> None

  3. 设置属性

    def shmem_set_attributes(my_rank, n_ranks, local_mem_size, ip_port, attributes) -> int

    参数/返回值

    含义

    my_rank

    int类型，当前秩

    n_ranks

    int类型，秩的总数

    local_mem_size

    int类型，当前占用的共享内存大小

    ip_port

    str类型，服务器的IP和端口号，例如tcp://ip:port

    attributes

    InitAttr类型，设置的属性

    返回值

    成功返回0，其他为错误码

  4. 查询共享内存模块的当前初始化状态

    def shmem_init_status() -> InitStatus

    参数/返回值

    含义

    返回值

    返回初始化状态。返回 SHMEM_STATUS_IS_INITIALIZED 表示初始化已完成

  5. 修改将用于初始化的属性中的数据操作引擎类型

    含义

    attributes

    InitAttr类型，属性集

    value

    返回值

    def register_decrypt_handler(py_decrypt_func:Callable[[str], str]) -> None

    参数/返回值

    含义

    py_decrypt_func

    可调用对象，一个Python函数，接受 (str cipher_text)，并返回 (str plain_text)

    cipher_text

    加密文本

    plain_text

    解密后的文本

    返回值

    成功时返回0,失败时返回错误代码

  8. 分配内存

    含义
